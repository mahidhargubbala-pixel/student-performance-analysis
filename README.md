## 📂 Projects

### 1. Data Cleaning & Visualization
`student_performance_analysis.ipynb`

Cleaned a messy student dataset — handling missing values, outliers, and inconsistent labels — then visualized key patterns with Matplotlib and Seaborn.

### 2. Exploratory Data Analysis (EDA)
`eda_project.ipynb`

Statistical summaries, correlation analysis, and visual pattern discovery to uncover what actually drives student performance.

### 3. Predictive Modeling with Machine Learning
`predictive_modelling_ml.ipynb`

Trained and compared Logistic Regression, Decision Tree, and Random Forest models to predict student pass/fail outcomes. Achieved 97.5% accuracy after debugging a data-quality issue that initially caused worse-than-random performance.

### 4. Real-World Data Project: Credit Card Fraud Detection
`fraud_detection_project.ipynb`

Built a fraud detection model on 284,807 real, anonymized credit card transactions (Kaggle: ULB Machine Learning Group), where fraud made up just 0.17% of the data. Compared Logistic Regression and Random Forest, evaluated using Precision, Recall, and F1-score instead of misleading accuracy metrics.

**Best model:** Random Forest — 96.1% Precision, 75.5% Recall, F1 = 0.85
